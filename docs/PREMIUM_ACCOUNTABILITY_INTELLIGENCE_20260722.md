# Premium Accountability Intelligence

The original Pro insights screen exposed only three descriptive values. This pass turns it into a premium accountability analysis product while preserving CalledOut's consequence-first positioning.

## Included analysis

- Last-30-day consistency with comparison to the prior 30 days
- Six-week kept-versus-missed momentum chart
- Current and longest streak
- Average proof-submission lead time
- Strongest weekday, workout type, and deadline window
- Weak-day detection with minimum sample protection
- Deterministic coaching guidance based on the user's actual record
- Redemption completion and open-redemption discipline
- Data-confidence meter so small samples are never overstated

## Data integrity

- Only resolved original promises count toward consistency.
- Redemption workouts are excluded from completion-rate calculations.
- Redemption never removes or converts the original miss.
- Weakest-day analysis requires at least two resolved promises on that weekday.
- The interface labels small samples as a building baseline or early read.
